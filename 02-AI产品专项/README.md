# AI 产品专项

> AI 产品有自己独特的设计挑战和机会

## 学习目标

掌握 AI 产品区别于传统产品的设计模式、用户体验挑战和评估方法。

## AI 产品设计模式

| 模式 | 典型产品 | 核心交互 |
|------|---------|----------|
| 对话式 | ChatGPT、Claude | 自然语言输入 → 文本输出 |
| 生成式 | Midjourney、Suno | 描述输入 → 创意内容输出 |
| 辅助式 | Copilot、Grammarly | 用户操作 → AI 实时建议 |
| Agent式 | Devin、Manus | 目标输入 → AI 自主完成任务 |
| 搜索增强式 | Perplexity | 问题输入 → 带引用的结构化回答 |

## AI 产品的独特挑战

1. **非确定性输出** —— 同样的输入不一定得到同样的结果
2. **质量难以保证** —— 不能像传统软件那样 100% 测试覆盖
3. **成本与质量的矛盾** —— 更好的模型 = 更高的成本
4. **用户预期管理** —— 用户容易高估或低估 AI 的能力
5. **快速变化的技术底座** —— 模型能力每几个月就有大幅提升
