# 认知升级

> 底层思维模型决定了你能走多远

## 学习目标

建立合伙人级别的思维框架和底层能力。

## 子目录

### 思维模型/
- **第一性原理** —— 回到最基本的事实，而不是类比和经验
- **系统思维** —— 看到反馈回路和非线性效应
- **逆向思维** —— 从"怎么会失败"反推"该怎么做"
- **概率思维** —— 在不确定性中做决策
- **杠杆思维** —— 找到投入产出比最高的事情

### 沟通表达/
- 结构化表达：金字塔原理
- 讲故事能力：把复杂的事说简单
- 说服力：如何推动一个决策被接受
- 写作能力：清晰的书面表达

### 行业认知/
- AI 技术演进趋势判断
- AI 行业的竞争格局
- 监管政策与合规
- 国内外市场差异
